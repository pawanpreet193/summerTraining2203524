Day 6: Methods and Functions

Introduced methods in Java, including syntax, parameters, return types, and the concept of method overloading. Practiced creating and using functions in Java.


public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }

Day 8: Constructors and 'this' Keyword

Explored constructors in Java, including the default and parameterized constructors. Also discussed the 'this' keyword for referring to current instance variables.


class Student {
    String name;
    int age;

    public Student(String name, int age) { // Parameterized Constructor
        this.name = name;
        this.age = age;
    }

    public void display() {
        System.out.println("Name: " + name + ", Age: " + age);
    }
}

Instantiated objects using constructors and observed the usage of 'this' for clarity.


Student student1 = new Student("John", 20);
student1.display();

Day 9: Inheritance in Java

Learned about inheritance in Java, a key OOP concept that allows one class to inherit properties and methods from another. Discussed the 'extends' keyword.


class Animal {
    public void eat() {
        System.out.println("This animal eats food.");
    }
}

class Dog extends Animal {
    public void bark() {
        System.out.println("The dog barks.");
    }
}

Created an instance of the subclass and demonstrated inheritance of methods.


Dog myDog = new Dog();
myDog.eat();
myDog.bark();

Day 10: Polymorphism and Method Overriding

Explored polymorphism and the concept of method overriding, where subclasses can provide specific implementations for methods defined in their superclass.


class Animal {
    public void sound() {
        System.out.println("Some sound...");
    }
}

class Cat extends Animal {
    @Override
    public void sound() {
        System.out.println("Meow");
    }
}

Demonstrated runtime polymorphism with overridden methods in subclasses.


Animal myCat = new Cat();
myCat.sound(); // Outputs: Meow

Day 11: Encapsulation and Access Modifiers

Covered encapsulation, a principle of hiding data and providing controlled access via getters and setters. Discussed access modifiers: private, public, protected.


class BankAccount {
    private double balance;

    public double getBalance() {
        return balance;
    }

    public void setBalance(double balance) {
        if (balance >= 0) {
            this.balance = balance;
        }
    }
}

Practiced using getter and setter methods to manage access to private variables.


BankAccount account = new BankAccount();
account.setBalance(500);
System.out.println("Balance: " + account.getBalance());

Day 12: Abstract Classes and Interfaces

Learned about abstract classes and interfaces in Java. Explored how abstract classes are used when we have a common base but want some methods to be implemented by subclasses. Interfaces were also covered as a way to define methods that multiple classes can implement.


abstract class Shape {
    abstract void draw();
}

class Circle extends Shape {
    @Override
    void draw() {
        System.out.println("Drawing Circle");
    }
}

interface Drawable {
    void draw();
}

class Rectangle implements Drawable {
    public void draw() {
        System.out.println("Drawing Rectangle");
    }
}

Used an interface to define methods across multiple classes.


Shape circle = new Circle();
circle.draw();

Drawable rectangle = new Rectangle();
rectangle.draw();

Day 20: Final Project - Notepad Application

Developed a fully functional Notepad application using Java and Swing. The project incorporates file handling, undo/redo functionality, dark mode, and text customization options.

Project Features:

  • Text Area with Line Numbering: A custom component was created to display line numbers alongside the text area for easy navigation in large files.
  • Undo/Redo Functionality: Implemented using Java's UndoManager, allowing users to undo and redo text changes.
  • File Operations: Features to create new files, open existing files, and save files, with JFileChooser for file selection and BufferedReader/BufferedWriter for reading and writing text.
  • Text Formatting: Users can modify font, font size, font color, and background color, as well as apply text styles like bold, italic, or plain.
  • Dark Mode: A toggle to switch between light and dark mode for the application interface.

Code Example:


import javax.swing.*;
import javax.swing.event.DocumentEvent;
import javax.swing.event.DocumentListener;
import javax.swing.undo.UndoManager;
import java.awt.*;
import java.awt.event.*;
import java.io.*;

public class NotepadApp extends JFrame implements ActionListener {
    JTextArea textArea;
    LineNumberingTextArea lineNumbers;
    UndoManager undoManager;
    
    public NotepadApp() {
        textArea = new JTextArea();
        lineNumbers = new LineNumberingTextArea(textArea);
        JScrollPane scrollPane = new JScrollPane(textArea);
        scrollPane.setRowHeaderView(lineNumbers);
        add(scrollPane);
        
        undoManager = new UndoManager();
        textArea.getDocument().addUndoableEditListener(undoManager);

        // Setup the menu and actions (not shown for brevity)
    }

    public void actionPerformed(ActionEvent e) {
        String command = e.getActionCommand();
        switch (command) {
            case "New": textArea.setText(""); break;
            case "Open": openFile(); break;
            case "Save": saveFile(); break;
            // More action handlers
        }
    }

    private void openFile() {
        JFileChooser fileChooser = new JFileChooser();
        int option = fileChooser.showOpenDialog(this);
        if (option == JFileChooser.APPROVE_OPTION) {
            File file = fileChooser.getSelectedFile();
            try (BufferedReader reader = new BufferedReader(new FileReader(file))) {
                textArea.read(reader, null);
            } catch (IOException ex) {
                ex.printStackTrace();
            }
        }
    }

    private void saveFile() {
        JFileChooser fileChooser = new JFileChooser();
        int option = fileChooser.showSaveDialog(this);
        if (option == JFileChooser.APPROVE_OPTION) {
            File file = fileChooser.getSelectedFile();
            try (BufferedWriter writer = new BufferedWriter(new FileWriter(file))) {
                textArea.write(writer);
            } catch (IOException ex) {
                ex.printStackTrace();
            }
        }
    }

    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> new NotepadApp().setVisible(true));
    }
}

The application allows users to work with text files efficiently, with features like line numbering, dark mode, and customizable text formatting options.
